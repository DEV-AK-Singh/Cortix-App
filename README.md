**Cortix -- AI-Native DevOps Automation Platform**
=================================================

Cortix is an **AI-powered DevOps automation system** that transforms any GitHub repository into **production-ready infrastructure**. With a single GitHub link, Cortix analyzes the entire codebase, identifies the tech stack, visualizes the project structure, and automatically generates YAML configs, Dockerfiles, and Bash setup scripts --- all powered by **Gemini 2.5 Flash**.
